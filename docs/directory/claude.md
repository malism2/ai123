# Claude

Claude is a family of large language models developed by Anthropic. Anthropic was founded by former members of OpenAI and focuses on AI safety and research.

## Key Features
- **Long Context Window**: Supports up to 200k context window.
- **Honesty & Safety**: Designed to be more helpful, honest, and harmless.
- **Artifacts**: Real-time preview of code, websites, and diagrams.

## Access
[Visit Claude](https://claude.ai)

## Pricing
- **Free**: Daily usage limits.
- **Pro**: $20/month, higher usage limits.
