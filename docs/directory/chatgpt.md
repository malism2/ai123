# ChatGPT

![ChatGPT](https://upload.wikimedia.org/wikipedia/commons/0/04/ChatGPT_logo.svg)

ChatGPT 是由 OpenAI 开发的人工智能聊天机器人程序，于 2022 年 11 月推出。该程序使用基于 GPT-3.5、GPT-4 架构的大型语言模型并利用强化学习训练。

## 核心功能
- **自然语言处理**：能够理解和生成类似人类的文本。
- **多模态支持**：支持图片识别、语音对话和 DALL-E 3 绘图。
- **代码编写**：辅助开发者编写、调试和优化代码。
- **插件系统**：通过 GPTs 扩展功能。
